# Nessie Example

[Project Nessie](https://projectnessie.org/) provides Git-like version control for your data lakehouse - branching, tagging, and time travel for Iceberg tables.

## Services

| Service | Port | URL |
|---------|------|-----|
| Floe | 9091 | http://localhost:9091 |
| Nessie | 19120 | http://localhost:19120 |
| SeaweedFS S3 | 18333 | http://localhost:18333 |
| Livy | 8998 | http://localhost:8998 |
| Trino | 8085 | http://localhost:8085 (with `-trino` target) |
