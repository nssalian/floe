# REST Catalog Example

The [Iceberg REST Catalog](https://iceberg.apache.org/rest-catalog-spec/) provides a standard HTTP interface for catalog operations.

## Services

| Service | Port | URL                                       |
|---------|------|-------------------------------------------|
| Floe | 9091 | http://localhost:9091                     |
| REST Catalog | 8181 | http://localhost:8181                     |
| MinIO Console | 9001 | http://localhost:9001 (admin/password)    |
| Livy | 8998 | http://localhost:8998                     |
| Trino | 8085 | http://localhost:8085 (with `admin` user) |
